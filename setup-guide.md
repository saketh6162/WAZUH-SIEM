# Setup Guide

## Step 1: Install Wazuh Manager (Ubuntu)

* Download and run installation script

## Step 2: Install Wazuh Agent (Windows)

* Install MSI package
* Provide manager IP

## Step 3: Register Agent

* Generate key on manager
* Import key on agent

## Step 4: Start Agent

* Restart Wazuh agent service

## Step 5: Verify Connection

* Check agent status in dashboard

## Step 6: Configure FIM

* Modify ossec.conf
* Add monitored directory
* Restart agent
